# Semantic Role Tags

Use exactly one of these 13 tags for each line's `semantic_role`. Do not invent new tags.

| Tag | When to use |
|---|---|
| 建立时间锚点 | Sets a time background |
| 建立场景 | Sets spatial or environmental context |
| 铺垫状态 | Describes current state, prepares for what follows |
| 深化状态 | Continues and deepens the buildup |
| 局部收束 | Closes a small sub-section |
| 情感转折 | Emotion or attitude shifts |
| 抬升 | Energy or emotion pushes upward |
| 主题宣言 | Directly states the core theme |
| Hook | Title sentence or core memory anchor |
| 回返 | Repeats or varies earlier content |
| 扩展确认 | Further reinforces an established theme |
| 收束 | Closes the full song or a major section |
| 过渡 | Bridges two segments |
