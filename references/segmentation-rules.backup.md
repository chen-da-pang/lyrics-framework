# Segmentation Rules

## Three-layer structure

- 句 (line) — smallest unit
- 段 (segment) — structural functional unit
- 轮次 (round) — full-song organizational unit

## Signal strength

Signals are additive. Cut when: 1 strong signal, OR 2+ medium signals, OR 1 medium + 2 weak signals.

**Strong signal (1 = cut):**
- Hook enters for the first time
- Hook restarts clearly as a new local beginning
- Transition from non-Hook zone to Hook zone
- Exit from Hook zone to closing section
- Clear terminal/closing sentence appears
- Sentence type switches dramatically (e.g. narrative → stutter/repeat pattern like 誰等等等等)

**Medium signal (2 = cut):**
- Repeat anchor restarts
- Sentence rhythm clearly restarts
- Structural task switches (e.g. buildup → lift → landing)
- One parallel group ends, another begins
- Syllable/character count pattern changes systematically across consecutive lines
- A clearly symmetric parallel group ends (e.g. 4 lines on theme A followed by 4 lines on theme B with identical sentence structure)

**Weak signal (do NOT cut alone):**
- Only semantic detail change
- Only imagery change
- Only diction style change
- Only slight mood change

## Segment roles (open set)

主歌 / 预副歌 / 副歌一 / 副歌二 / 扩展副歌 / 结尾 / 桥段 / 后副歌

## Key rules

**主歌:**
- 主歌 does not split into two by default
- BUT: if the verse contains two clearly symmetric parallel groups (e.g. 4 lines on "圆" + 4 lines on "完" with identical sentence structure), treat them as 主歌一 and 主歌二
- The test: can you swap the two groups and the song still makes structural sense? If yes, they are two separate segments

**预副歌:**
- 预副歌 is a group of lift sentences, NOT a single sentence
- A round can start directly with 预副歌 — not every round needs a 主歌 (common in second-round returns)

**副歌:**
- If the chorus has two distinct opening lines that alternate across repetitions (e.g. 「情人团聚的小心愿」and「成为遗憾的小心愿」), treat them as 副歌一 and 副歌二 — not the same chorus repeated
- 副歌一 and 副歌二 always appear as a pair; if one is present, look for the other
- Do not merge 副歌一 and 副歌二 into one segment just because they share similar lines in the middle

**桥段:**
- 桥段 is typically 1-3 lines that break the established pattern entirely
- Often features the longest lines in the song, or a sudden shift to spoken/prose style
- Does NOT need to be long — even a single distinctive line can be 桥段 if it structurally separates two chorus blocks

**轮次:**
- A round does not need to contain all segment types
- Common patterns:
  - R1: 主歌一 → 主歌二 → 预副歌 → 副歌一 → 副歌二
  - R2: 预副歌 → 副歌一 → 副歌二 (no verse — compressed return)
  - R3: 桥段 → 副歌一变体 → 副歌二变体 (outro)
- The number of segments is a judgment result, not a preset template

## Segmentation process

1. **Read the full lyrics first** — identify all repeated blocks before cutting anything
2. **Find all Hook anchors** — mark every line that appears to open a chorus
3. **Check for dual-Hook pattern** — if two different lines each open a chorus block, label them 副歌一 and 副歌二
4. **Identify verse structure** — check if the verse has internal symmetry (parallel groups); if so, split
5. **Find round boundaries** — where does the song restart its structural cycle?
6. **Name segments** — round first, then role (e.g. 第一轮·主歌一, 第一轮·副歌二)
